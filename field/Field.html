# 三、字段

# 3.1 字段属性

一个字段具备以下属性：

属性名 类型 含义 备注
id ID 一个字段定义的唯一ID 唯一标识
name String 标识该字段的唯一名字 符合[a-z]+
type String 类型名 类型名可以是标量类型，也可以是自定义类型的类型的引用
notnull Boolean 是否可以为空 标识该字段是否可以为空
default String 字段默认值 默认值的JSON表述，如果为引用类型，则为引用对象的ID
islist boolean 是否为List 表示该字段是否为某类型的数组,默认为false
isunique boolean 是否唯一字段 如果是唯一字段，将会为该字段创建唯一索引
issystemfield boolean 是否系统字段 表示该字段为系统保留字段，默认为false，用户无法定义一个系统字段
fieldattr Enum 字段属性 scalartype、reference

通过操作界面只能添加userdefined字段，systemfield字段时系统自定义字段，会在创建相关类型时自动赋给相关自定义类型。
