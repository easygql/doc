# 三、字段

# 3.1 字段属性

一个字段具备以下属性：

属性名 类型 含义 备注
id ID 一个字段定义的唯一ID 唯一标识
name String 标识该字段的唯一名字 符合[a-z]+
type String 类型名 类型名可以是标量类型，也可以是自定义类型的类型的引用
notnull Boolean 是否可以为空 标识该字段是否可以为空
default String 字段默认值 默认值的JSON表述，如果为引用类型，则为引用对象的ID
fieldkind Enum 字段类型 systemfield、userdefined
fieldattr Enum 字段属性 scalartype，reference

注：

  1. 通过操作界面只能添加userdefined字段，systemfield字段时系统自定义字段，会在创建相关类型时自动赋给相关自定义类型。

  2. scalartype表示标量，reference表示引用。

# 3.2 标量类型

标量类型主要有：

类型名 说明
Int 整形
Long 长整型
Short 短整形
Float 浮点型
BigDecimal 数值型
ID 唯一ID(自定义字段暂不支持该类型)
Boolean 布尔型
String 字符串
Byte 字节型
BigInteger 大整数
Char 字符型
Object 对象型
Json JSON对象型
Date 日期型
DateTime 时间型
Time 时间型
CreatedAt 创建时间，用户不可修改，系统自动为该字段的赋值。
LastUpdate 修改时间，用户不可修改，系统自动为该字段赋值。
