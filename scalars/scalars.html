# 二、标量类型

# 2.1 ID

表示这是一个唯一主键，在EasyGQL中，默认且强制每一个数据对象中，字段"id"的数据类型为“ID“，使用者不可以定义其他字段属性为“ID”，如果对象定义中没有"id"字段，EasyGQL默认会增加该字段。

# 2.2 Float

有符号双精度浮点值。

# 2.3 Boolean

布尔型数据，True或者False

# 2.4 String

UTF‐8 字符序列

# 2.5 Int

有符号32位整数

# 2.6 Long

有符号64位整数

# 2.7 Short

有符号16位整数

# 2.8 Byte

字节

# 2.9 BigInteger

大整形

# 2.10 BigDecimal

BigDecimal

# 2.11 Char

字符

# 2.12 Object

任意Java对象

# 2.13 Json

任意Json字符串

# 2.14 Date

格式为yyy-MM-dd的日期

# 2.15 DateTime

ISO_OFFSET_DATE_TIME 格式的时间
