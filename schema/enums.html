# 四、枚举类型

本章主要介绍现阶段EasyGQL枚举类型的定义、使用场景、用法.

# 4.1 枚举类型

枚举场景主要适用于定义一些值域只有固定几个可选值的字段，比如性别、国籍、省份、学历等。 枚举类型主要通过定义枚举类型元素据以及定义枚举值来完成，其中枚举类型元数据主要有以下属性：

序号 属性 含义 非空 默认值
1 ID 唯一标识 是 无
2 name 类型名 是 无
3 schema 所属Schema 是 无

枚举类型的枚举值主要有以下属性：

序号 属性 含义 非空 默认值
1 id 唯一标识 是 无
2 value 值 是 无
3 enum 所属枚举类型 是 无

# 4.2 枚举类型API

在EasyGQL控制台中提供如下跟枚举类型相关的API

# 4.2.1 新增

在API中，enumtypesInSchema_IDInput和enumtypesInSchema_ObjectInput是在Schema中增加枚举类型，valuesInEnumType_IDInput和valuesInEnumType_ObjectInput是给枚举类型添加枚举值

input EnumTypeInput   {
  description: String = ""
  id: ID
  name: String!
}

type EnumType_InsertResult  {
  id_list: String
  nodes: [EnumType]
}
type EnumType  {
  id: ID
  name: String
  schemaid: [SchemaObject]
  values: EnumElement
}
type EnumElement  {
  description: String
  enumtype: [EnumType]
  id: ID
  value: String
}
enumtypesInSchema_IDInput(from_id: ID!, reset: Boolean = false, to_id: [ID]!): _NestInputResult 
enumtypesInSchema_ObjectInput(from_id: ID!, reset: Boolean = false, to_object: [EnumTypeInput]!): _NestInputResult 
valuesInEnumType_IDInput(from_id: ID!, reset: Boolean = false, to_id: [ID]!): _NestInputResult 
valuesInEnumType_ObjectInput(from_id: ID!, reset: Boolean = false, to_object: [EnumElementInput]!): _NestInputResult

# 4.2.2 修改

在EasyGQL控制台中，EnumTypeUpdate是修改枚举类型，EnumElementUpdate是修改枚举元素值

EnumTypeUpdate(object: EnumTypeUpdateInput!, where: EnumTypeWhereInput): _UpdateResult
EnumElementUpdate(object: EnumElementUpdateInput!, where: EnumElementWhereInput): _UpdateResult

# 4.2.3 删除

在EasyGQL控制台中，valuesInEnumTypeRemove是删除一个枚举类型的具体枚举元素，enumtypesInSchemaObjectRemove是删除一个Schema中的一个具体枚举类型，

valuesInEnumTypeRemove(child_Id: [ID], parent_Id: ID!): EnumElement_DeleteResult
enumtypesInSchemaObjectRemove(child_Id: [ID], parent_Id: ID!): EnumType_DeleteResult

# 4.3 枚举类型的使用

# 4.3.1 枚举类型字段

一个枚举类型字段主要有以下属性：

序号 属性名 含义 默认值 是否非空
1 id 字段的唯一标识 系统会自动赋予每个新增字段一个唯一标识 非空
2 name 字段的名称 无 非空
3 type 字段的类型 无 非空
4 islist 是否为列表 否 非空
5 notnull 字段是否非空 否 非空
6 invisible_roles 不可查看的角色列表 空List 非空
7 irrevisible_roles 不可查看的角色列表 空List 非空
8 description 描述 "" 可空
9 defaultvalue 默认值 可空

# 4.3.2 枚举字段API

在EasyGQL,通过以下定义在一个对象中增加和删除枚举字段

input EnumFieldInput  {
  defaultvalue: Object
  id: ID
  invisible_roles: [[Role]]
  irrevisible_roles: [[Role]]
  islist: Boolean = false
  name: String!
  type: String! 
}
enumfieldsInContentType_IDInput(from_id: ID!, reset: Boolean = false, to_id: [ID]!): _NestInputResult 
enumfieldsInContentType_NestRemove(from_id: [ID]!, to_id: ID!): _NestRemoveResult 
enumfieldsInContentType_ObjectInput(from_id: ID!, reset: Boolean = false, to_object: [EnumFieldInput]!): _NestInputResult
