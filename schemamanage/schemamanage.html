# 五、Schema的管理

# 5.1 登录管理后台

在http://ip:port/login 界面下 输入账号密码，目前是admin,admin,进入schema管理后台。 可以在Schema管理后台对Schema进行管理。整个后台管理页面如下图所示：

# 5.2 Schema的创建

点击“New Schema”按钮，如下图所示，输入Schema的名字、自定义的数据类型、所基于的数据库类型（目前只支持rethinkdb和mongodb），注意同一个用户下Schema的名字不能重复。

点击保存，如果Schema创建成功，会返回Save Success。

# 5.3 Schema的修改

在Schema列表中选择对应Schema中的Edit按钮，可以对自定义数据类型进行修改。

点击保存，如果成功会返回“Save Success”，失败会返回对应错误。在进行Schema修改时，Schema中已经有的数据会保持原样，新增的字段为空，删除的字段不会被查询到。

# 5.4 Schema的删除

在Schema列表中选择对应Schema中的Delete按钮，可以对Schema进行删除。

# 5.5 Schema定义下载

在Schema列表中选择对应Schema中的Download按钮，可以下载EasyGQL根据用户定义的类型生成的graphql chema。

# 5.6 DataExplorer

在Schema列表中选择对应Schema中的Data Explorer超链接，可以测试生成的Schema。
