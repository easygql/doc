# 七、 Restful API

# 7.1 api/graphql/{schemaid}

该API同时支持Get和Post方法，接收参数类型为"application/json;charset=UTF-8",返回参数类型为“application/json;charset=UTF-8”

# 7.1.1 输入参数

输入参数 说明
schemaid PathVariable，表示执行操作的Schema
operationName 操作名，非必填
query 必填项，表示查询语句
variables 变量值，非必填

EasyGQL会根据用户输入query语句判断，具体操作为Query、Mutation、Subscription。EasyGQL的Restful API不提供订阅功能，所有的subscription操作都会转换为查询操作。

# 7.1.2 输出参数

输出参数是查询结果主要包含以下字段：

输出参数 说明
data 操作成功时的结果
error 操作失败时的具体错误内容
