# 四、注解

# 4.1 index

这是一个基于自定义对象级的注解类型，使用方法为：

type chatroom @index(indexinfo:{indexname:"testindex",indexcol="id,owner"}){
	id:ID
	owner:String!
}

表示基于id和owner两个属性建立复合索引。

# 4.2 lastupdate

这是一个属性级的注解，表示该字段是一个时间戳字段，每次进行数据修改时，该字段都会自动更新。使用方法如下：

type chatroom @index(indexinfo:{indexname:"testindex",indexcol="id,owner"}){
	id:ID
	owner:String!
	lastupdatetime:DateTime@lastupdate
}

所注解的属性，数据类型必须是DateTime。

# 4.3 createdat

这是一个属性级的注解，表示该字段是一个时间戳字段，每次进行数据新增时，该字段都会自动赋值。使用方法如下：

type chatroom @index(indexinfo:{indexname:"testindex",indexcol="id,owner"}){
	id:ID
	owner:String!
	createdat:DateTime@creatdat
}

所注解的属性，数据类型必须是DateTime。
