# 四、枚举类型

本章主要介绍现阶段EasyGQL枚举类型的定义、使用场景、用法.

# 4.1 枚举类型

枚举场景主要适用于定义一些值域只有固定几个可选值的字段，比如性别、国籍、省份、学历等。 枚举类型主要通过定义枚举类型元素据以及定义枚举值来完成，其中枚举类型元数据主要有以下属性：

序号 属性 含义 非空 默认值
1 ID 唯一标识 是 无
2 name 类型名 是 无
3 schema 所属Schema 是 无

枚举类型的枚举值主要有以下属性：

序号 属性 含义 非空 默认值
1 id 唯一标识 是 无
2 value 值 是 无
3 enum 所属枚举类型 是 无

# 4.3 枚举类型的使用

# 4.3.1 枚举类型字段

一个枚举类型字段主要有以下属性：

序号 属性名 含义 默认值 是否非空
1 id 字段的唯一标识 系统会自动赋予每个新增字段一个唯一标识 非空
2 name 字段的名称 无 非空
3 type 字段的类型 无 非空
4 islist 是否为列表 否 非空
5 notnull 字段是否非空 否 非空
6 invisible_role 不可查看的角色列表 空List 非空
7 unmodifiable_role 不可查看的角色列表 空List 非空
8 description 描述 "" 可空
9 defaultvalue 默认值 可空
